# research-agent - 调研智能体

自动完成竞品调研全流程的智能代理。

## 能力

- 自主决策采集策略
- 分析市场格局
- 识别创作机会
- 生成调研报告

## 触发条件

- 用户请求进行竞品调研
- 用户询问某个领域的内容机会
- 需要批量分析视频数据

## 工作流程

```
1. 理解需求
   ↓
2. 制定采集策略（关键词、数量、筛选条件）
   ↓
3. 执行数据采集
   ↓
4. 分析数据模式
   ↓
5. 识别创作机会
   ↓
6. 生成调研报告
   ↓
7. 给出行动建议
```

## 使用示例

**用户**：帮我调研一下"老人养生"这个领域，看看有什么内容机会

**Agent 执行**：
1. 分析"老人养生"相关关键词
2. 采集 1000+ 相关视频
3. 分析市场格局（频道集中度、播放量分布）
4. 识别机会（小频道爆款、高互动模板）
5. 生成调研报告
6. 输出 3-5 条可执行的内容建议

## 输出格式

```markdown
## 调研结论：老人养生领域

### 市场概况
- 样本量：1,020 个视频
- 总播放量：50M+
- 活跃频道：412 个

### 关键发现
1. 标题含"秘诀"的视频播放量高 2.3 倍
2. 5-10 分钟时长表现最好
3. Top10 频道占 45% 播放量

### 创作机会
1. 小频道爆款（23 个）：可模仿的成功模板
2. 近期热门（8 个）：当前市场热点
3. 高互动模板（15 个）：内容质量标杆

### 行动建议
1. 参考视频 A 的标题结构
2. 控制时长在 5-8 分钟
3. 关注"XXX"关键词
```

## 调用方式

```python
from agents.research_agent import ResearchAgent

agent = ResearchAgent()
result = agent.run(
    topic="老人养生",
    target_count=1000,
    auto_report=True
)
```
