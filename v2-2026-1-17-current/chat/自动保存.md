> zed 中的文档，可以设置为自动保存吗？

⏺ 是的，Zed 编辑器支持自动保存功能。

  设置方法

  打开 Zed 设置文件（Cmd + , 或菜单 Zed > Settings > Open Settings），添加以下配置：

  {
    "autosave": {
      "after_delay": {
        "milliseconds": 1000
      }
    }
  }

  可用选项

  | 选项                                   | 说明           |
  |--------------------------------------|--------------|
  | "off"                                | 关闭自动保存       |
  | {"after_delay": {"milliseconds": N}} | 停止输入 N 毫秒后保存 |
  | "on_focus_change"                    | 失去焦点时保存      |
  | "on_window_change"                   | 切换窗口时保存      |

  常用配置示例

  {
    "autosave": "on_focus_change"
  }

  这样当你切换到其他文件或窗口时会自动保存。

> Zed > Settings > Open Settings这里打开的不是配置文件

⏺ 你说得对。在 Zed 中正确的打开方式：

  打开设置文件

  快捷键：Cmd + ,

  或者通过命令面板：
  1. Cmd + Shift + P 打开命令面板
  2. 输入 open settings 或 zed: open settings
  3. 选择 Zed: Open Settings

  这会打开 ~/.config/zed/settings.json 文件。

  添加自动保存配置

  在 JSON 文件中添加：

  {
    "autosave": {
      "after_delay": {
        "milliseconds": 1000
      }
    }
  }

  如果文件已有其他配置，把 "autosave" 加到现有的大括号内即可。

> 怎么验证配置成功了？

⏺ 最简单的验证方法：

  1. 打开任意文件，随便修改一些内容
  2. 观察标签页标题 — 修改后会显示一个小圆点 ● 表示未保存
  3. 等待你设置的延迟时间（比如 1 秒）
  4. 小圆点消失 = 自动保存成功

  如果配置了 "on_focus_change"：
  - 修改文件后，点击另一个文件或窗口
  - 小圆点消失 = 配置生效

> "milliseconds": 1000是多久？

⏺ 1000 毫秒 = 1 秒

  常用换算：

  | 毫秒值  | 时间    |
  |------|-------|
  | 500  | 0.5 秒 |
  | 1000 | 1 秒   |
  | 2000 | 2 秒   |
  | 3000 | 3 秒   |

  根据你的习惯调整，比如想要停止输入 2 秒后再保存，就设置 "milliseconds": 2000。
