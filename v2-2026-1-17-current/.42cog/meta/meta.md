## 项目描述

面向 YouTube 内容创作者的**全流程辅助工具**，核心价值是帮助创作者**找到好选题、了解竞争格局、优化自己的频道**。

不同于简单的数据罗列工具，本项目提供：
1. **有分析框架**（四象限、时长-播放矩阵）而非只展示原始数据
2. **可解释的 AI 洞察**（推理链、置信度、数据源关联）
3. **Actionable**（每个洞察都对应具体行动建议）

## 核心功能模块

| 模块 | 功能 | 解决的问题 |
|------|------|-----------|
| **选题发现** | 输入关键词，分析该领域 1000+ 视频 | "我该做什么内容？" |
| **竞品分析** | 视频榜单、频道榜单、模式洞察 | "这个领域谁做得好？怎么做的？" |
| **频道诊断** | 输入频道链接，获得评分和改进建议 | "我的频道哪里需要改进？" |
| **监控追踪** | 关键词监控、趋势追踪、定时采集 | "市场有什么新变化？" |
| **发布辅助** | 批量上传、定时发布 | "如何高效发布视频？" |

## 目标用户

主要面向**中文 YouTube 创作者**，特别是：
- 做养生、教程、美食、旅行等垂类内容
- 目标受众在新加坡、马来西亚、台湾、香港等华语地区
- 想通过数据驱动来提升内容表现

## 核心价值主张

### 数据分析维度

**视频榜单：**
- 🔥 爆款视频（高播放量）
- ⚡ 近期爆款（新发布但增长快）
- 🌲 长青视频（老视频但日均播放高）
- 📈 潜力视频（被低估的内容）
- ⏱️ 长视频榜

**频道榜单：**
- 📊 总播放榜
- ⚡ 平均播放榜
- 🐴 黑马榜（粉丝少但表现好）
- 🚀 高效榜

**筛选维度：**
播放量、地区、时长、粉丝数、发布时间、互动率、AI视频识别

### 可视化设计

#### 1. 内容四象限 (Content Map)

```
        高互动率
           ↑
    💎 粉丝向   ⭐ 爆款型
    (Niche)      (Star)
           ↑
低播放 ←——————————————→ 高播放
           ↓
    ❄️ 冷门型   🚀 破圈型
    (Dog)       (Viral)
           ↓
        低互动率
```

把**播放量 × 互动率**两个维度交叉，产生四种内容类型：
- **Star**：学习标杆
- **Niche**：深耕细分
- **Viral**：优化互动
- **Dog**：避免重复

#### 2. 可解释 AI 洞察系统

```
┌─────────────────────────────────────────────┐
│ 💡 洞察标题                    [展开/折叠]  │
├─────────────────────────────────────────────┤
│ [置信度条 ████████░░ 78%]                   │
│ 数据源: [视频数据] [频道数据] [趋势数据]     │
├─────────────────────────────────────────────┤
│ ┌─────────────┐  ┌─────────────────────┐   │
│ │  可视化图表  │  │    推理链          │   │
│ │  (SVG/图)   │  │  📊 观察 → 权重30% │   │
│ │             │  │      ↓             │   │
│ │             │  │  📈 分析 → 权重50% │   │
│ │             │  │      ↓             │   │
│ │             │  │  ✅ 结论           │   │
│ └─────────────┘  └─────────────────────┘   │
└─────────────────────────────────────────────┘
```

核心设计：
- **置信度条**：用颜色（高/中/低）表示洞察可靠性
- **数据源标签**：点击可高亮关联数据
- **推理链展示**：把 AI 得出结论的过程拆解成步骤
- **权重条**：显示每个推理步骤的贡献度
- **可交互高亮**：源数据 ↔ 洞察 ↔ 依赖项之间联动高亮

#### 3. 时长分布可视化

```
0-1分钟   ████░░░░░░ 供给 12%
1-3分钟   ██████░░░░ 供给 28%
3-5分钟   ████████░░ 供给 15%   ← 平均播放最高
5-10分钟  ██████████ 供给 8%
```

### 套利分析框架

```
有趣度 = 信息价值程度 / 信息传播程度 = 中介中心性 / 程度中心性
```

高有趣度 = 价值被低估 = 套利机会

| 套利类型 | 发现方法 | 适用博主 |
|----------|----------|----------|
| 话题套利 | 关键词网络中介中心性 | 腰部博主 |
| 跨语言套利 | 源市场价值/目标市场供给 | 头部博主 |
| 时长套利 | 时长分桶有趣度 | 小白博主 |
| 频道套利 | 小频道爆款分析 | 小白博主 |
| 趋势套利 | 话题频率变化率 | 头部博主 |

## 技术栈

**前端**:
- HTML/CSS/JS 单页应用
- Chart.js 图表库
- 暗色主题 UI

**后端**:
- Python ≥3.10
- FastAPI (WebSocket 实时通信)
- yt-dlp 视频数据采集

**自动化层**:
- Playwright 浏览器自动化
- 定时调度任务

**核心工具**:
- 视频处理: yt-dlp, FFmpeg
- 语音识别: Whisper
- 运行时: Node.js (bun)
