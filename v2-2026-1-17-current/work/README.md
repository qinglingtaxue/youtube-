1. **按层级阅读**：先看战略层决策理解方向，再看架构层理解技术选型
2. **按时间阅读**：按日期顺序阅读可还原决策演进过程
3. **按需查阅**：遇到具体问题时查阅对应的决策文档了解背景
4. **回溯决策**：通过来源文件 ID 可追溯原始对话上下文
