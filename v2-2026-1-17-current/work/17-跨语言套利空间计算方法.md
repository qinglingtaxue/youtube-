## 决策背景 / Decision Background

### 问题识别

1. 中文市场已有大量内容，英语/日语市场内容稀缺
2. 如何**量化**翻译/重制一个话题的价值？
3. 需要排序：先做「太极」还是「穴位按摩」？

### 触发因素

- 模式21发现英语市场均播是日语的1.5-8倍
- 需要决策：进入哪个市场？翻译哪个话题？

## 决策内容 / Decision Content

### 核心公式

```
套利空间 = (源市场内容量 / 目标市场内容量) × (目标市场均播 / 源市场均播)
```

### 公式推导

```
套利空间 = 内容稀缺度 × 表现提升度

其中：
- 内容稀缺度 = 源市场内容量 / 目标市场内容量
  → 源市场内容丰富(可翻译)但目标市场内容少 → 稀缺度高

- 表现提升度 = 目标市场均播 / 源市场均播
  → 翻译后能获得的播放量提升倍数
```

### 计算示例

#### 示例1：太极 中文→英语

```
源市场（中文）：90条视频，均播7.8万
目标市场（英语）：20条视频，均播217万

套利空间 = (90/20) × (217/7.8)
         = 4.5 × 27.8
         = 125.1

解读：极高套利空间
- 中文有90条可翻译的内容素材
- 翻译后均播预期提升28倍
- 英语市场仅20条竞品
```

#### 示例2：穴位按摩 中文→英语

```
源市场（中文）：101条视频，均播4万
目标市场（英语）：0条视频，均播未知（假设为中文均播×2=8万）

套利空间 = (101/0.1) × (8/4)  # 0条按0.1计算避免除零
         = 1010 × 2
         = 2020

解读：极高套利空间
- 英语市场完全空白
- 中文有101条可翻译内容
- 先发优势巨大
```

#### 示例3：八段锦 中文→英语

```
源市场（中文）：20条视频，均播143万
目标市场（英语）：35条视频，均播229万

套利空间 = (20/35) × (229/143)
         = 0.57 × 1.6
         = 0.9

解读：低套利空间
- 英语市场已有更多内容(35条>20条)
- 均播提升仅1.6倍
- 不建议进入
```

## 博主套利空间公式 / Creator Arbitrage

除了话题级别的套利，还可以计算**单个博主**的翻译价值：

```
博主套利有趣度 = (话题多样性 × 均播质量) / (总传播度)
              = 内容生产能力 / 已有知名度
```

### 博主评估示例

| 博主 | 视频数 | 均播 | 订阅 | 效率(均播/订阅) | 有趣度 | 建议 |
|------|--------|------|------|-----------------|--------|------|
| 养生老郑 | 4 | 41.5万 | 1.8万 | 22.9 | **0.23** | 翻译价值最高 |
| laxnetcm | 3 | 27.3万 | 0.5万 | 51.5 | **0.20** | 效率极高 |
| 太极芸庄 | 3 | 119万 | 12万 | 9.9 | 0.15 | 均播高但知名度也高 |

**高套利博主特征**：
- 视频少（3-5条）但均播高（>10万）
- 播放效率极高（均播/订阅 > 20）
- 尚未被广泛关注（订阅<2万）

## 综合收益公式 / Combined Revenue Formula

结合 TD-18 的 CPM 数据，完整的收益潜力公式为：

```
收益潜力 = 套利空间 × 目标市场CPM × 爆款率 / 竞争度

其中：
- 套利空间 = (源内容量/目标内容量) × (目标均播/源均播)
- CPM = 见 TD-18
- 爆款率 = 百万播放视频占比
- 竞争度 = 目标市场现有视频数
```

## 局限性 / Limitations

1. **假设均播可复制**：实际上翻译后的表现可能不同
2. **未考虑制作成本**：翻译/配音/本地化的成本未纳入
3. **市场动态变化**：套利空间会随时间缩小

### 改进方向

- 引入「制作成本系数」
- 引入「市场饱和速度」预估
- 定期更新数据重新计算
