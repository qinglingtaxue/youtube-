# 决策11：调整爆款 Top50 判定标准

> 产品层决策 | 2026-01-20 | 来源：556a11bc

## 决策背景

### 问题识别
- 原有爆款判定标准过于简单
- Top50 排名需要更合理的算法
- 需要排除异常数据干扰

### 触发因素
- 发现部分频道数据缺失
- 排名结果与预期不符
- 需要更精细的排名机制

## 决策内容

### 核心决策
**调整爆款 Top50 的判定标准，优化排名算法**

### 原有标准
```
排名 = 按播放量降序排列
爆款 = 播放量 Top 50
```

### 新标准

**1. 数据质量过滤**
- 剔除频道数据缺失的视频
- 剔除播放量异常波动的视频
- 剔除重复或搬运内容

**2. 综合评分算法**
```
综合得分 = 播放量权重 × 0.4
         + 互动率权重 × 0.3
         + 增长速度权重 × 0.2
         + 内容质量权重 × 0.1
```

**3. 分时间段排名**
| 时间段 | 标准 | 用途 |
|--------|------|------|
| 24h Top50 | 近24小时表现 | 发现新爆款 |
| 7d Top50 | 近7天表现 | 周度热门 |
| 30d Top50 | 近30天表现 | 月度榜单 |
| 历史 Top50 | 累计表现 | 经典爆款 |

### 排名展示
```
┌─────────────────────────────────────────┐
│ 爆款 Top 50 (7天)                        │
├────┬───────────────────┬────────┬───────┤
│ #  │ 视频标题          │ 播放量  │ 评分  │
├────┼───────────────────┼────────┼───────┤
│ 1  │ 视频A             │ 500万  │ 98.5  │
│ 2  │ 视频B             │ 450万  │ 96.2  │
│ ...│ ...               │ ...    │ ...   │
└────┴───────────────────┴────────┴───────┘
```

## 决策理由

### 支持因素
- 综合评分更全面反映视频价值
- 数据过滤提高榜单质量
- 分时间段满足不同需求

### 变更原因
- 单一播放量排名易被刷量干扰
- 频道缺失导致数据不完整
- 用户需要多维度榜单

## 影响范围

### 直接影响
- 修改排名算法逻辑
- 更新前端榜单展示
