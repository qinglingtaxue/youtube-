## 决策背景 / Decision Background

### 问题识别

1. 发现了24个模式，但**哪些模式更有价值**？
2. 如何区分「众所周知的常识」和「真正有价值的洞察」？
3. 需要一个**客观的评估标准**来排序模式优先级

### 触发因素

- 模式1「头部效应」人人都知道，但模式22「互动率-播放量负相关」很少有人知道
- 需要量化「有趣程度」来指导行动优先级

## 决策内容 / Decision Content

### 核心公式

```
有趣度 = 信息价值 / 信息传播度 ≈ 中介中心性 / 程度中心性
```

### 理论依据：网络科学

本框架借鉴了**网络科学**中的中心性概念：

| 概念 | 网络科学定义 | 在信息评估中的含义 |
|------|-------------|-------------------|
| **中介中心性（Betweenness Centrality）** | 一个节点作为最短路径中介的次数 | 信息连接不同知识领域的能力 |
| **程度中心性（Degree Centrality）** | 一个节点的直接连接数 | 信息已被传播/引用的广度 |

### 公式推导

```
有趣度 = 桥梁价值 / 已知程度
       = 能连接多少不同决策 / 有多少人已经知道
       ≈ 中介中心性 / 程度中心性
```

**直觉解释**：
- 一个信息如果能帮助很多不同决策（高中介），但很少人知道（低程度）→ 极有趣
- 一个信息人人都知道（高程度），但只能帮助一种决策（低中介）→ 无聊

### 评分方法

| 维度 | 评分 | 定义 |
|------|------|------|
| **信息价值** | 1-5分 | 能指导多少不同决策？连接多少策略领域？ |
| **传播程度** | 1-5分 | 有多少人已经知道？是否为常识？ |

### 有趣度分类

| 类型 | 价值 | 传播度 | 有趣度 | 特征 |
|------|------|--------|--------|------|
| 🔴 高价值低传播 | 高 | 低 | **极高** | 反直觉、颠覆认知 |
| 🟡 高价值高传播 | 高 | 高 | 中等 | 常识但验证有价值 |
| ⚪ 低价值高传播 | 低 | 高 | **极低** | 众所周知的废话 |
| ⚫ 低价值低传播 | 低 | 低 | 低 | 无意义噪声 |

## 应用示例 / Application Examples

### 模式22：互动率-播放量负相关

```
信息价值 = 5
  - 改变内容策略核心认知
  - 连接：播放预测、互动设计、用户分层、KPI设计

传播程度 = 1
  - 几乎无人知道这个规律
  - 完全反直觉（普遍认为高互动=高播放）

有趣度 = 5 / 1 = 5.0（极高）
```

### 模式1：头部效应幂律分布

```
信息价值 = 2
  - 只能验证数据质量
  - 无法指导差异化策略

传播程度 = 5
  - 二八法则人人都知道
  - 是基础常识

有趣度 = 2 / 5 = 0.4（极低）
```

## 局限性 / Limitations

1. **主观性**：价值和传播度的评分依赖主观判断
2. **领域依赖**：不同领域的「常识」不同
3. **时效性**：随着传播，有趣度会下降

### 改进方向

- 通过问卷调查量化「传播程度」
- 通过引用网络量化「信息价值」
- 定期更新评分
