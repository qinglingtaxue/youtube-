## 核心交互设计

### 1️⃣ 维度切换（优先级：视频 > 搜索关键词 > 频道 > 标题词 > 国家）

```
用户点击「[○ 搜索关键词]」或「[○ 频道]」等
  ↓
当前显示的「有趣度榜」、「中介中心性榜」、「程度中心性榜」
都切换到对应维度
  ↓
排序和筛选条件保留不变
```

### 2️⃣ 排序控制

```
左侧：榜单类型（不变）
[● 💎有趣度]  [○ 🌉中介]  [○ 🔥程度]

右侧：排序控制
时间: [过去 3 个月 ▼]
字段: [有趣度 ▼]
方向: [高→低 ▼]

当前规则显示: 过去 3 个月，按有趣度高→低排序 [▶ 展开筛选]
```

### 3️⃣ 四象限参考图（帮助用户理解）

显示在三个榜单下方，帮助用户理解：
- 💎 有趣度高 + 🔥 竞争低 = 套利区（新手机会）
- 🌉 中介中心性高 = 跨领域连接机会

## 关键设计说明

### 为什么加搜索关键词维度？

```
视频维度：看内容本身的价值（直接选题参考）
搜索关键词维度：看用户搜索需求（选题+内容缺口发现）
频道维度：看频道运营样本（学习如何做频道）
标题词维度：看标题创意素材（学习怎么起标题）
国家维度：看跨语言机会（拓展到其他语言市场）
```

搜索关键词维度特别重要，因为：
1. **搜索需求 = 内容缺口** → 穴位按摩搜索 +76%，但内容稀缺
2. **跨域连接** → 某个搜索词能连接多个用户群体
3. **竞争分析** → 某个热词竞争激烈度（搜索关键词程度中心性）
