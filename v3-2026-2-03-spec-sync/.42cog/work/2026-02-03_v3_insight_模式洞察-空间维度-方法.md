## 模式20：地区差异分析

### 方法

1. **地区分组**：按 `channels.country` 分组
2. **聚合指标**：
   - 频道数：`COUNT(DISTINCT channel_id)`
   - 视频数：`COUNT(*)`
   - 均播放：`AVG(view_count)`
   - 最高播放：`MAX(view_count)`
3. **供需缺口识别**：
   - 供给：YouTube视频数量
   - 需求：Google Trends搜索热度
   - 缺口 = 高搜索热度 + 低视频数量

### 偏差风险

- 频道国家覆盖率仅21%，代表性不足
- 单个爆款可能扭曲均值（如加拿大的八段锦爆款）

### 置信度判定

**高**（已验证），但需扩大覆盖率至80%+

## 模式21：多语言市场对比

### 方法

1. **语言识别**：基于采集时的语言参数（en/ja）
2. **分组聚合**：按语言分组，计算均播放、最高播放
3. **话题×语言交叉**：同一话题在不同语言的表现对比
4. **差异倍数**：`英语均播 / 日语均播`

### 偏差风险

- 样本量较小（英语81+日语91=172条）
- 采集关键词可能有语言偏差

### 置信度判定

**高**（N=172，跨语言验证完整）

## 模式24：综合收益潜力评估

### 核心公式

```
收益潜力 = 预期播放量 × CPM × 爆款率 × 竞争系数
```

| 变量 | 计算方式 |
|------|----------|
| 预期播放量 | 该话题×语言的历史均播放 |
| CPM | 外部数据源（按国家） |
| 爆款率 | 百万+视频占比 |
| 竞争系数 | 1 / 现有视频数（视频越少竞争越低） |

### 收益估算模型

```
单视频收益 = 播放量 / 1000 × CPM

示例：10万播放 × $12 CPM = $1,200
```

### 爆款率计算

```
爆款率 = COUNT(view_count >= 1000000) / COUNT(*)
```

### 偏差风险

- CPM为行业平均值，个体差异大
- 爆款率基于历史数据，未来可能变化

### 置信度判定

**高**（CPM为公开数据，视频数据已验证）
