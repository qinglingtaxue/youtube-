# 模式洞察 - 时间维度 - 数据层

> 包含模式：5, 13, 15
> 最后更新：2026-01-24

---

## 数据表

| 数据表 | 用途 |
|--------|------|
| competitor_videos | 视频发布时间、播放量统计 |

## 关键字段

| 字段 | 类型 | 说明 |
|------|------|------|
| timestamp | UNIX时间戳 | 视频发布时间（秒级精度） |
| title | VARCHAR | 视频标题（用于话题识别） |
| view_count | INT | 播放量 |
| published_at | DATE | 发布日期 |

## 数据范围

| 项目 | 说明 |
|------|------|
| 时间范围 | 2007-2026（跨度19年） |
| 有效分析区间 | 2024-01 ~ 2026-01（数据密集期，2024前数据稀疏） |
| 视频总数 | 2340 条 |

## 数据质量

| 问题 | 状态 | 影响 |
|------|------|------|
| 2024前数据稀疏 | ⚠️ 已知 | 早期趋势分析不可靠 |
| 话题识别用标题关键词匹配 | ⚠️ 存在误分类风险 | 话题归类可能不准确 |
| timestamp字段精度 | ✅ 秒级 | 支持小时级分析 |

## 采集命令

```sql
-- 按星期统计发布效果
SELECT
    DAYOFWEEK(FROM_UNIXTIME(timestamp)) as day_of_week,
    COUNT(*) as video_count,
    AVG(view_count) as avg_views,
    MAX(view_count) as max_views
FROM competitor_videos
WHERE timestamp IS NOT NULL
GROUP BY day_of_week
ORDER BY avg_views DESC;

-- 话题生命周期查询
SELECT
    topic,
    MIN(published_at) as first_appear,
    MAX(published_at) as last_appear,
    DATEDIFF(MAX(published_at), MIN(published_at)) as span_days,
    COUNT(*) as video_count,
    SUM(view_count) as total_views
FROM competitor_videos
GROUP BY topic;
```
