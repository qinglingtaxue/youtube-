## 模式11：频道稳定性分析

### 方法

1. **稳定性指标**：`max/avg比 = MAX(view_count) / AVG(view_count)`
2. **分类标准**：
   - max/avg < 5：稳定
   - max/avg 5-10：较稳定
   - max/avg 10-20：不稳定
   - max/avg > 20：极不稳定
3. **筛选条件**：视频数 ≥ 5（保证统计意义）

### 解读

- 比值越小 → 播放量越稳定 → 粉丝粘性高
- 比值越大 → 依赖单个爆款 → 不可持续

### 偏差风险

- 新频道样本量小，比值不稳定
- 不同内容类型的正常比值范围可能不同

### 置信度判定

**高**（Top 10 频道分析）

## 模式12：黑马频道识别

### 方法

1. **爆发倍率**：`burst_ratio = MAX(view_count) / subscriber_count`
2. **排序**：按爆发倍率降序
3. **筛选**：订阅数 > 0

### 解读

- 高爆发倍率 = 单视频播放远超订阅数
- 黑马特征：订阅1-5万，但单视频播放百万+

### 偏差风险

- 低订阅频道容易出现高比值（分母小）
- 需结合绝对播放量判断

### 置信度判定

**高**（Top 10 黑马分析）

## 模式14：快速增长频道分析

### 方法

1. **筛选条件**：
   - 活跃期 ≤ 30天：`DATEDIFF(MAX(published_at), MIN(published_at)) <= 30`
   - 订阅数 ≥ 1000
   - 每条视频 ≥ 1000播放
2. **指标计算**：
   - 活跃天数
   - 视频数
   - 总播放
   - 平均播放

### 偏差说明

```
⚠️ 偏差1：无法验证「一个月涨1000粉」
────────────────────────────────────────
数据库中没有「频道创建时间」字段
只能用「首个视频发布日」近似，可能：
• 频道创建后很久才发第一个视频
• 之前的视频已删除
• 是老频道转型

✅ 解决方案：
需要调用 YouTube API 获取 channel.snippet.publishedAt
或使用第三方工具（Social Blade）验证
```

```
⚠️ 偏差2：采样偏差
─────────────────
只采集了「养生」领域的视频
这些快速增长频道可能是养生领域特有的
不能推广到科技/游戏/娱乐等其他领域
```

```
⚠️ 偏差3：可能是矩阵账号
─────────────────
laxne / laxnetcm / Laxne 疑似同一人运营
总订阅：5300 + 23100 + 91800 = 12万+
实际是成熟运营者，非「新手快速增长」
```
