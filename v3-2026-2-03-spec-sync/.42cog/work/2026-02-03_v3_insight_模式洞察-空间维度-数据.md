# 模式洞察 - 空间维度 - 数据层

> 包含模式：20, 21, 24
> 最后更新：2026-01-24

---

## 数据表

| 数据表 | 用途 |
|--------|------|
| channels | 频道国家信息 |
| competitor_videos | 视频播放量、语言标识 |

## 关键字段

| 字段 | 类型 | 说明 |
|------|------|------|
| channels.country | VARCHAR | 频道所在国家（覆盖率21%，208/979） |
| competitor_videos.view_count | INT | 播放量 |
| competitor_videos.language | VARCHAR | 视频语言（en/ja/zh等） |

## 外部数据源

| 数据源 | 用途 | 覆盖范围 |
|--------|------|----------|
| Google Trends API | 搜索热度 | TW/US/HK/JP/KR |
| YouTube CPM公开数据 | 收益估算 | 全球主要市场 |

### CPM数据来源

| 国家 | CPM(美元) | 数据来源 |
|------|-----------|----------|
| 挪威 | $43.15 | isthischannelmonetized.com |
| 澳大利亚 | $36.21 | awisee.com |
| 美国 | $12-14 | lenostube.com |
| 新加坡 | $17.75 | 同上 |
| 香港 | $17.23 | 同上 |
| 日本 | $5.68 | 同上 |
| 台湾 | $3-4 | 同上 |
| 印度 | $0.74 | 同上 |

## 数据范围

| 项目 | 说明 |
|------|------|
| 频道国家覆盖率 | 21%（208/979频道） |
| 多语言视频样本 | 英语81条 + 日语91条 = 172条 |
| Google Trends地区 | TW/US/HK/JP/KR 五地区 |

## 数据质量

| 问题 | 状态 | 影响 |
|------|------|------|
| 频道国家覆盖率低(21%) | ⚠️ 需扩大 | 地区分析代表性不足 |
| 多语言样本量较小(172) | ⚠️ 可用 | 跨语言对比有一定可信度 |
| CPM为外部估算数据 | ℹ️ 已知 | 实际收益可能有偏差 |

## 采集命令

```bash
# 多语言视频采集
python3 scripts/collect_multilang.py --lang en ja ko --max 50

# 多语言搜索热度
python3 scripts/fetch_trends.py --multilang

# 分析多语言数据
python3 scripts/collect_multilang.py --analyze

# 频道国家信息补全
python3 scripts/fetch_channel_info.py
```
