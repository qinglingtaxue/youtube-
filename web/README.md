# 功能演示页面

## 打开演示页面

### 方法1: 直接在浏览器中打开
```bash
# 在浏览器中打开
open demo.html
# 或直接双击 demo.html 文件
```

### 方法2: 通过Web服务器访问
```bash
# 启动Web服务器（已启动）
python3 -m http.server 8080 --bind 127.0.0.1

# 在浏览器中访问
# http://localhost:8080/demo.html
```

- 展示项目统计数据（8+示例脚本、20+地区、4个平台）

- 快速执行

#### 调研工具

- 实时预览命令行

- 查看低竞争地区

#### 平台选择
- YouTube

- 8个可用示例脚本

### 4. 快速开始

- 42COG框架

## 交互功能

1. **标签切换**: 点击不同的标签查看不同内容
2. **地区选择**: 点击地区标签进行选择/取消选择
3. **命令生成**: 输入品类和选择地区后，实时生成命令
4. **地区分组查看**: 点击按钮查看不同分组的地区
