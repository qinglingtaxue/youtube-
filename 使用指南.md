## 快速上手

### 第一步：理解框架

本项目基于**三事件最小故事**框架：

```
事件1：调研分析 → 产出典型案例 + 模式总结
事件2：模式抽象 → 生成创作模板
事件3：实战应用 → 输出创作文案
```

### 第二步：确定调研主题

选择一个具体的主题，例如：
- "老人养生"
- "AI工具推荐"
- "美食制作"
- "健身教程"

### 第三步：准备工具

确保您有以下工具：
- 网络连接（可访问YouTube）
- 文本编辑器（记录分析结果）
- 数据表格（整理案例信息）

## 工作流详解

### 事件1：调研分析（60分钟）

#### 阶段1：数据收集（20分钟）

**步骤1：关键词扩展**
```
主关键词：老人养生
↓
扩展关键词：
- 中医养生
- 健康科普
- 长寿秘诀
- 保健方法
- 养生误区
```

**步骤2：快速搜索**
- 每个关键词搜索50-100个视频
- 重点关注播放量 > 1000的视频
- 筛选发布时间在30天内

**步骤3：初步筛选**
- 去除明显低质量内容
- 过滤重复视频
- 保留10个高质量案例

## 模板应用

### 模板库

#### 1. 认知冲击型模板

**模板A：数据震撼型**
```
开头：你以为[常识观点]？
其实[颠覆数据]的人都在[做某事]

发展：根据[权威]调查，[具体数字]
的人发现[真相]

专家：[专家姓名]表示，[核心观点]

结尾：关于[话题]，你怎么看？
关注我，每天分享[相关]干货
```

**模板B：对比反差型**
```
开头：[群体A]和[群体B]的[方面]
差距竟然这么大！

发展：[数据对比]显示
- [群体A]：[表现1]
- [群体B]：[表现2]

解释：原因在于[关键因素]

结尾：你属于哪一类？评论区说说
```

#### 2. 故事叙述型模板

**模板A：经历分享型**
```
开头：去年这个时候，我[遇到某事]
没想到[转折]

发展：原本以为[预期]
没想到[意外情况]

转折：后来我[尝试某方法]
终于[结果]

总结：这件事让我明白[道理]
```

**模板B：案例故事型**
```
开头：我有个朋友[背景描述]

发展：他遇到了[问题]
试了很多方法都不行

转折：后来用了[解决方案]
效果特别好

结尾：如果你也遇到[类似问题]
不妨试试[建议]
```

### 模板使用技巧

#### 1. 个性化调整
- 根据领域调整语言风格
- 根据受众调整专业程度
- 根据平台调整内容长度

#### 2. 元素替换
- 保持结构，替换具体内容
- 保留句式，更换关键词
- 维持节奏，调整表达方式

#### 3. 效果验证
- 小范围测试效果
- 根据反馈优化调整
- 记录最佳实践

## 高级技巧

### 1. 模式组合使用

**组合策略**
```
认知冲击 + 故事叙述
效果：震撼开场 + 情感共鸣
适用：需要快速吸引注意并建立连接

干货输出 + 互动引导
效果：价值传递 + 用户参与
适用：专业内容推广

案例展示 + 数据支撑
效果：真实可信 + 权威背书
适用：产品或服务推广
```

### 2. 平台适配

**YouTube适配**
- 视频时长：3-15分钟
- 标题长度：60字符以内
- 缩略图：鲜艳色彩+人物出镜
- 描述：详细+关键词+链接

**抖音适配**
- 视频时长：15-60秒
- 标题：简短+话题标签
- 开头：3秒内抓住注意力
- 节奏：快速切换+字幕辅助

**B站适配**
- 视频时长：5-20分钟
- 标题：详细+系列化
- 封面：精美设计+标题文字
- 互动：弹幕+评论区引导

### 3. 数据追踪

**关键指标**
- 播放量增长率
- 点赞率/评论率
- 完播率
- 转化率（关注/分享）

## 常见问题

### Q1：如何判断案例质量？

**判断标准**
- 播放量：符合领域平均水平的2倍以上
- 互动率：点赞数/播放量 ≥ 3%
- 内容质量：信息密度高、逻辑清晰
- 创新性：有独特观点或表达方式

### Q2：模式识别不准确怎么办？

**解决方案**
1. 增加分析样本数量
2. 细化分析维度
3. 对比多个相似案例
4. 寻求专家意见

### Q3：模板使用效果不佳？

**优化方法**
1. 检查是否适合目标受众
2. 调整语言风格和表达方式
3. 优化内容结构和节奏
4. 增加个性化和创新元素

### Q4：如何避免内容同质化？

**差异化策略**
1. 结合个人经历和观点
2. 使用独特的表达方式
3. 融入行业前沿动态
4. 创造专属的视觉风格

### Q5：如何持续优化工作流？

**优化方向**
1. 记录每次执行的时间和效果
2. 收集用户反馈和建议
3. 分析成功案例的共同要素
4. 定期更新模板库和案例库
