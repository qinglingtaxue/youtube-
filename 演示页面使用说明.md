# 可视化演示页面使用说明

## 🎯 演示页面地址

项目已创建可视化演示页面，展示所有功能：

```
web/demo.html
```

## 🚀 如何查看

### 方法1: 直接打开文件
```bash
# 在浏览器中打开
open web/demo.html
```

### 方法2: 使用Web服务器
```bash
cd web
python3 -m http.server 8080

# 然后在浏览器中访问
# http://localhost:8080/demo.html
```

### 方法3: 使用启动脚本
```bash
cd web
./open_demo.sh
```

## 🎮 交互功能

### 标签切换
点击不同的标签查看不同内容：
- 调研工具
- 地区管理
- 平台选择
- 示例脚本

### 地区选择
点击地区标签进行选择/取消选择：
- 选中的地区会高亮显示
- 支持多地区同时选择

### 命令生成
输入品类和选择地区后，点击"生成命令"按钮：
- 实时生成完整的命令行
- 显示MCP调用命令
- 支持滚动查看完整内容

### 地区分组查看
点击按钮查看不同分组的地区：
- 低竞争地区
- 中等竞争地区
- 高竞争地区
- 英语地区
