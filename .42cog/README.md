# YouTube视频的最小故事 - 42COG项目

## 项目概述

本项目基于**认知敏捷法 (42COG)** 构建，专注于YouTube视频创作的最小故事工作流。通过"事件1→事件2→事件3"的三事件结构，将复杂的调研工作简化为可执行的三步走策略。

## 核心工作流

### 最小故事框架
- **事件1**：调研x个视频 → 找到领域高手 + 总结创作模式
- **事件2**：逆向工程 → 将模式应用到文案创作
- **事件3**：实战创作 → 写出解决领域难题的创作文案
- **生发意义**：积累skill文档 → 成为领域高手

### 时间效率要求
- 单次调研：不超过2小时
- 数据分析：30分钟内完成模式抽象
- 模式转化：直接输出可执行的创作模板

## 核心实体

### 1. Researcher（调研员）
负责整个工作流的执行，从数据收集到模式抽象，再到创作转化。

### 2. Sample（视频样本）
被调研的视频对象，需要从1000个候选中筛选出10个高质量案例。

### 3. Pattern（创作模式）
从样本中抽象出的规律，如"认知冲击型"、"故事叙述型"等。

### 4. Template（创作模板）
可复用的创作框架，直接指导视频创作。

### 5. Workflow（工作流）
完整的三事件流程，确保调研工作高效有序。

## 关键优势

1. **高效性**：2小时内完成从调研到模板的全流程
2. **实用性**：输出可直接执行的创作指导
3. **可复用性**：积累的skill文档形成知识资产
4. **低门槛**：30分钟学习成本，工具链简单易用

## 质量标准

- 数据质量：10个高质量案例 > 1000个浅层数据
- 分析深度：必须抽象出可复用的创作模式
- 转化效率：调研结果必须直接指导创作
- 速度要求：比手工方式快5倍以上

## 工作流示例

```
输入：主题词（如"老人养生"）
↓
事件1：快速调研 → 找到3-5个典型案例 → 总结模式
↓
事件2：模式抽象 → 生成创作模板 → 验证可行性
↓
事件3：实战创作 → 输出文案 → 验证效果
↓
输出：完整的创作模板库
```
