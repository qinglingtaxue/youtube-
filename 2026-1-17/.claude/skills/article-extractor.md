# article-extractor

提取网页文章内容。

## 触发条件

用户要提取网页内容、获取文章正文时使用。

## 执行步骤

1. 使用 WebFetch 获取网页内容
2. 提取正文、标题、作者等
3. 转换为 Markdown 格式

## 使用方式

使用 WebFetch 工具，prompt 设置为：

```
请提取这个页面的文章内容，包括：
1. 标题
2. 作者（如果有）
3. 发布日期（如果有）
4. 正文内容（转为 Markdown 格式）
5. 关键要点总结
```

## 输出格式

```markdown
# {标题}

**作者**：{作者}
**日期**：{日期}
**来源**：{URL}

---

{正文内容}

---

## 要点总结

- 要点 1
- 要点 2
- ...
```

## 输出

- Markdown 文件：`data/articles/{日期}_{标题}.md`
