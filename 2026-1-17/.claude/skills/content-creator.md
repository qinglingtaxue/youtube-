# content-creator

品牌声音分析和 SEO 优化。

## 触发条件

- 用户要分析品牌声音
- 用户要优化 SEO
- 用户要检查内容一致性

## 功能

### 1. 品牌声音分析

分析现有内容，提取品牌声音特征：
- 语气（正式/轻松/专业/亲切）
- 用词风格
- 句式特点
- 常用表达

### 2. SEO 优化

优化视频元数据：
- 标题优化（50-70 字符，含关键词）
- 描述优化（前 150 字符含关键词）
- 标签推荐（5-15 个，混合高频+长尾）

### 3. 一致性检查

检查内容是否符合品牌声音：
- 对比现有风格
- 标记不一致的地方
- 提供修改建议

## 输出

### SEO 报告格式

```json
{
  "titles": [
    {"title": "...", "score": 85, "reason": "..."}
  ],
  "description": "...",
  "tags": ["标签1", "标签2"],
  "seo_score": 85,
  "suggestions": ["建议1", "建议2"]
}
```

## 输出位置

- SEO 报告：`scripts/seo_report_{date}.json`
- 品牌声音分析：`data/reports/brand_voice.md`
