# ideation

创意构思和头脑风暴。

## 触发条件

- 用户要进行头脑风暴
- 用户要生成创意点子
- 用户需要内容灵感

## 执行步骤

1. 理解主题和目标受众
2. 从多个维度发散思考
3. 评估每个点子的可行性
4. 推荐 Top 3

## 思考维度

### 话题角度
- 常规 vs 新奇
- 正面 vs 反面
- 专业 vs 入门
- 理论 vs 实践

### 内容形式
- 教程演示
- 对比测评
- 问答解惑
- 故事叙述
- 挑战实验
- 清单盘点

### 差异化
- 竞品未覆盖的角度
- 独特的表达方式
- 创新的结构设计

## 输出格式

```markdown
# 创意清单：{主题}

## 点子列表

### 1. {点子标题}
- **角度**：{切入角度}
- **形式**：{内容形式}
- **可行性**：⭐⭐⭐⭐ (4/5)
- **差异化**：{独特之处}
- **预估效果**：{为什么可能有效}

### 2. ...

## Top 3 推荐

1. **{点子}** - {推荐理由}
2. **{点子}** - {推荐理由}
3. **{点子}** - {推荐理由}
```

## 输出

- 创意清单：`data/reports/ideation_{date}.md`
