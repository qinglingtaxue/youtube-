# 学术信息分析报告 - 通用模板

> 基于《Journal of Financial Economics》信息分析报告提炼
> 适用于：学术期刊分析、学科领域分析、研究主题分析

---

## 一、模板概览

### 报告结构（结论前置型）

```
1. 项目背景        → 建立认知基础
2. 结论摘要        → 核心洞察先行（2-3个）
3. 发展历史        → 时间线梳理
4. 重要人物        → 关键学者网络
5. 重要知识        → 核心概念与演变
6. 研究趋势        → 未来方向预测
7. 附录清单        → 数据支撑
8. 分析思路        → 方法论复盘
```

## 二、数据需求清单

### 2.1 原始数据来源

| 数据类型 | 推荐来源 | 用途 |
|----------|----------|------|
| 文献元数据 | WOS、Scopus、Dimensions | 核心分析数据 |
| 引用数据 | WOS、Google Scholar | 影响力分析 |
| 反驳/支持数据 | Scite | 争议性分析 |
| 影响力指标 | SJR、JCR、CiteScore | 地位评估 |
| 社交传播数据 | Altmetric | 有趣度计算 |

### 2.2 需要采集的字段

**文献层面：**
- 标题（Title）
- 作者（Authors）
- 发表年份（Year）
- 摘要（Abstract）
- 关键词（Keywords）- 作者关键词 + 索引关键词
- 引用数（Citations）
- 参考文献（References）
- 机构（Affiliations）
- 国家（Countries）
- DOI

**期刊/领域层面：**
- 创刊时间
- 出版频率
- 研究领域分类
- H指数
- 影响因子
- 奖项设置
- 主编信息

## 四、核心分析框架

### 4.1 时间维度分析

| 分析项 | 方法 | 产出 |
|--------|------|------|
| 发文量变化 | 年度统计 | 折线图 |
| 引用量变化 | 年度统计 | 折线图 |
| 主题演变 | Thematic Evolution | 桑基图 |
| 阶段划分 | 拐点识别 | 时间线 |

**分析要点：**
- 识别增长/衰退拐点
- 关联外部事件（政策、危机、技术变革）
- 与出版频率、主编更迭等内部事件关联

### 4.2 人物维度分析

| 分析项 | 方法 | 指标 |
|--------|------|------|
| 高产学者 | 发文量排名 | 文献数 |
| 高影响学者 | 被引排名 | 总引用/H指数 |
| 高价值学者 | 中介中心性 | Betweenness |
| 有趣学者 | 价值/传播比 | Betweenness/Citations |
| 学者群组 | 共被引分析 | Co-citation Network |
| 合作关系 | 合作网络 | Collaboration Network |
| 师承关系 | 定性调研 | 导师-学生链 |

**分析要点：**
- 各阶段核心人物是谁？
- 形成了哪些学术流派/学派？
- 高产≠高影响≠高价值，需区分
- 关注"有趣学者"：价值高但传播低

### 4.3 知识维度分析

| 分析项 | 方法 | 产出 |
|--------|------|------|
| 高频主题 | 词频统计 | 词云/TreeMap |
| 主题关系 | 共现分析 | Co-occurrence Network |
| 主题分类 | Thematic Map | 四象限图 |
| 高被引文献 | 被引排名 | Top N 清单 |
| 高争议文献 | 反驳数排名 | Scite 数据 |
| 经典理论 | 定性分析 | 理论演变图 |

**Thematic Map 四象限解读：**
```
          高密度
            │
  成熟主题  │  核心主题
  (Motor)   │  (Basic)
────────────┼────────────  高中心性
  边缘主题  │  新兴主题
  (Niche)   │  (Emerging)
            │
          低密度
```

### 4.4 空间维度分析

| 分析项 | 方法 | 产出 |
|--------|------|------|
| 国家分布 | 作者国籍统计 | 地图可视化 |
| 机构分布 | 机构发文量 | 排名表 |
| 国际合作 | 合作网络 | 网络图 |

## 九、常见问题

### Q1: 数据不完整怎么办？
- 多数据源交叉补充
- 明确标注数据边界
- 缺失部分用定性分析补充

### Q2: 作者名有重复/变体？
- Bibliometrix 有作者名标准化功能
- 手动检查高频作者

### Q3: 关键词太分散？
- 使用 Keywords Plus（WOS自动生成）
- 手动合并同义词

### Q4: 如何计算"有趣度"？
```
有趣度 = 价值程度 / 传播程度
      = 中介中心性 / 被引数
```
高有趣度 = 价值高但传播低，值得深挖

### Q5: 分析周期要多久？
- 快速版：1-2周（核心指标）
- 标准版：3-4周（完整分析）
- 深度版：6-8周（含定性调研）
